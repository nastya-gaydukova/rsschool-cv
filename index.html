Coding is done as follows: s = 'aaaabbсaa' is converted to 'a4b2с1a2', that is, groups of identical characters in the original string are replaced with this character and the number of its repetitions in this position of the string.

s = input('')
previous = s[0]
a = 0
b = ''
for current in s:
    if previous == current:
        a = a + 1
    if previous != current:
        b = b + previous + str(a)
        a = 1
        previous = current
b = b + current + str(a)
print(b)
